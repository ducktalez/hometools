# hometools

Provides useful PC tools everyone will find handy.
Special features

- Music-library sanitizer
  - finding dupes
  - unifying file-names
  - updating meta-infos
- Movie-library sanitizer
  - collecting Movie meta-infos from `https://www.themoviedb.org/` (tmdb)
  - Evaluating the *correct* `tmdbid`, perfectly suiting Mediaservers like `JellyFin` or `Plex`  to prevent false media-assumptions.
